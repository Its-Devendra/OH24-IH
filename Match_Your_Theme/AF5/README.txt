THEME: Event Ticketing System

CHALLENGE-1: Easy and Generic 
Develop an event ticketing system that allows users to search for events, purchase tickets, and receive event updates using an event ticketing API (e.g., Eventbrite API).

------------------------------------------------------------------------------------
CHALLENGE-2: Web (HTML, CSS, JavaScript, Node.js)
Create a web-based event ticketing system with a backend server using Node.js, allowing event organizers to create and manage event listings, sell tickets online, and track attendance.

------------------------------------------------------------------------------------
CHALLENGE-3: Python (Flask or Django) 
Build a Python-based event ticketing system with a backend server using Flask or Django, implementing features such as user authentication, event promotion tools, and ticket scanning capabilities for event entry management.

------------------------------------------------------------------------------------
CHALLENGE-4: Desktop Event Ticketing Application
Develop a desktop event ticketing application using Java or CPP with a GUI framework, providing a user-friendly interface for browsing event listings, purchasing tickets, and managing event registrations. Integrate it with backend.
